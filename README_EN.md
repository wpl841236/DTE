# DTE
DTE – D DOS T Terminal E Emulator, short for DOS Terminal Emulator

# Compile Source Code
On a computer with GNU G++ and GNU Make installed, open the CMD terminal, navigate to the source code directory, and run the `make` command to complete compilation. The compiled output executable will be `main.exe`.
